## Templates

- [Full template](adr-template.md) — all sections with explanations
- [Minimal template](adr-template-minimal.md) — mandatory sections with explanations
- [Bare template](adr-template-bare.md) — all sections, empty
- [Bare minimal template](adr-template-bare-minimal.md) — mandatory sections, empty
